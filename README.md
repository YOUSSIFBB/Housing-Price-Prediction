🏠 Housing Price Prediction & Email Classification

This repository contains machine learning projects focused on applying AI-driven models and statistical algorithms to real-world datasets. The work demonstrates a structured approach to data analysis, predictive modeling, and performance evaluation across both regression and classification problems.

Overview

The project is built around two main use cases:

1. Housing Price Prediction
Developed a regression model using the Austin housing dataset to estimate property prices. This involved applying machine learning algorithms, feature selection techniques, and data preprocessing to improve prediction accuracy.

2. Email Spam Classification
Built a classification model to detect spam emails using text processing methods and supervised learning algorithms. The model evaluates patterns in email content to classify messages effectively.

Technical Approach:
* Applied machine learning algorithms (regression and classification models)
* Used data preprocessing and feature engineering techniques
* Performed exploratory data analysis to identify key patterns
* Evaluated model performance using standard metrics
* Developed reproducible workflows in Jupyter Notebooks
  
Tools & Technologies:
* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib / Seaborn
* Jupyter Notebook
